Установить интерпретатор Python

    Прямая ссылка на скачивание установщика для Windows 64-bit:
    https://www.python.org/ftp/python/3.10.2/python-3.10.2-amd64.exe

    При необходимости, выбрать другой установочный пакет на странице:
    https://www.python.org/downloads/release/python-3102/


Установить текстовый редактор Notepad++

    Прямая ссылка на скачивание установщика для Windows 64-bit:
    https://github.com/notepad-plus-plus/notepad-plus-plus/releases/download/v8.3.2/npp.8.3.2.Installer.x64.exe

    При необходимости, выбрать другой установочный пакет на странице:
    https://notepad-plus-plus.org/downloads/v8.3.2/

    Для OS X рекомендую Atom:
    https://atom.io/
