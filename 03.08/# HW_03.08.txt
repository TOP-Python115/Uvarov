Установить интерпретатор Python

Прямая ссылка на скачивание установщика для Windows 64-bit:
https://www.python.org/ftp/python/3.10.2/python-3.10.2-amd64.exe

При необходимости, выбрать другой установочный пакет на странице:
https://www.python.org/downloads/release/python-3102/



Установить текстовый редактор Notepad++

Прямая ссылка на скачивание установщика для Windows 64-bit:
https://github.com/notepad-plus-plus/notepad-plus-plus/releases/download/v8.3.2/npp.8.3.2.Installer.x64.exe

При необходимости, выбрать другой установочный пакет на странице:
https://notepad-plus-plus.org/downloads/v8.3.2/

В качестве ответа на ДЗ прикрепить снимок экрана с установленными приложениями


Код каждой решённой задачи помещается в текущий каталог репозитория в виде отдельного файла с именем, соответствующем номеру задачи.

Все файлы выполняются с помощью интерпретатора.
Текстовый вывод в стандартный поток stdout копируется.
Копия вывода в виде комментария помещается в конец файла с кодом задачи.

В сервис Журнал отправляется сообщение о выполнении задании со ссылкой на репозиторий.
